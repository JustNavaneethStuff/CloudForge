## Suggested Study Order

1. Service choices → understand the "what" and "why"
2. Terraform architecture → understand the "how"
3. Tradeoffs and alternatives → demonstrate critical thinking
4. Failure scenarios → show production mindset
5. Scaling + cost → show business awareness
6. Operational procedures → show day-2 readiness
